Create a Heroku project wit the Django starter template
1 - Create a virtual environment
$ virtualenv venv

2 - Pip install
Django and all other packages necessary for project

3 - Create Django project with the Heroku Django starter template
Note that the project name at the end of the line is for Django,
and not the name your project will be known as on Heroku
$ django-admin startproject --template=https://github.com/heroku/heroku-django-template/archive/master.zip --name=Procfile <PROJECT_NAME>

4 - Deploy to Heroku
Change SECRET_KEY to os.environ.get() first
$ git init
$ git add -A
$ git commit -m "Initial commit"

$ heroku create <HEROKU_NAME>
$ git push heroku master
$ heroku run python manage.py migrate

5 - Scale dynos if using worker and/or clock dynos
$ heroku scale worker=1 clock=1
$ heroku ps
